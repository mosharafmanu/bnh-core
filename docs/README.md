## Bensnaturalhealth Theme For Wordpress

This theme is based on underscore mininal theme for more info about underscore theme head over this link https://github.com/Automattic/_s 

## Configuration 
Simply Install the wordpress and uplaod this theme, you are ready to go. 

## Style and javascript editing

Bootstrap is used for gird and model popup.

To edit the style follow the following step

1. Download the Node, Node package manager and gulp gobally 
2. Make sure your working directory is wp-content/theme/bensnaturalhealth 
3. run npm install, will install all required dev dependencies. 
4. edit gulpfile.js as your requirement and run gulp watch to compile all scss file and see the changes live on browser(browsersync is used to sync the changes so you might need to change proxy setting depending on your local development environment).

editing javascript file 
----------------------
Open js/main.js.
Almost all javascript is code depends on jquery and bootstrap.js.
